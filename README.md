# Grails-hotels

Каталог отелей

## Описание

Сайт с каталогом отелей, находящихся в разных странах мира.

## Возможности
- Поиск отелей по каталогу с возможностью выбора страны
- Просмотр, добавление, редактирование, удаление отелей и стран
- Переход на сайт нужного отеля

## Технологии
- Groovy
- Grails 4
- H2 database
